## как алгоритмы из стратегий повышают эффективность агентов
-----
Microsoft предлагает использовать метод Monte Carlo Tree Search (MCTS), изначально применявшийся в стратегических играх, для построения ИИ-агентов на деревьях решений. 

При достаточном бюджете токенов и времени этот подход позволяет эффективно перебирать варианты, обеспечивая значительный прирост производительности в тестах.
