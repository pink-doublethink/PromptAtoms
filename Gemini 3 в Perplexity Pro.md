## новые возможности для архитектуры кода и интеграции с GitHub 
-----
Подписчикам Perplexity Pro стала доступна модель Gemini 3, которая, по утверждению автора, работает мощнее версии в AI Studio. 

Благодаря встроенному коннектору к GitHub, пользователи могут использовать сервис для архитектурного планирования и создания Development Plan. 

Предлагаемый рабочий процесс включает генерацию кода через Gemini Pro 2.5 и последующую отладку с помощью Grok Code Fast.
