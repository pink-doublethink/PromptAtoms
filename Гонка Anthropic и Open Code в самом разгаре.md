## Гонка Anthropic и Open Code в самом разгаре 
-----
Автор проводит параллель между текущим противостоянием сообщества Open Code и компании Anthropic с исторической борьбой Linux против Microsoft. 

В социальных сетях формируется образ нового «цифрового сопротивления» монополиям, копирующий риторику прошлых лет об «империи зла». 

Однако автор иронично отмечает цикличность этого процесса: подобные лозунги борьбы с «капиталистами» неизбежно будут сопровождать выход любых будущих крупных моделей ИИ.
