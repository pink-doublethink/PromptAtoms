## почему новый бенчмарк Coding Arena может стать стандартом оценки ИИ-программистов 
-----
LM Arena провела ребрендинг в Arena AI, представив важную новинку — Coding Arena. Этот инструмент эмулирует работу типичного ИИ-агента с реальным набором функций (list_dir, edit_file), приближая условия тестирования к средам вроде Claude Code или Cursor. 

Новый бенчмарк претендует на авторитетность уровня SWE-Bench, заполняя нишу тестирования моделей в контексте полноценного агента, что особенно актуально для задач фронтенд-разработки, где возможна валидная визуальная оценка результата.
