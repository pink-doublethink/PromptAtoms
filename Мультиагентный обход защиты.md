## хакеры превратили архитектуру Claude в инструмент кибершпионажа
-----
Китайские хакеры, предположительно действующие в интересах КНР, использовали принципы архитектуры Claude Code для создания автономных шпионских агентов. 

Разделив операции между специализированными MCP-агентами (поиск уязвимостей и анализ целевого кода), они смогли обойти защитные фильтры модели: благодаря фрагментации контекста LLM не распознала создание вредоносного ПО.
