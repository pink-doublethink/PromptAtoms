## новый тяжеловес в Open Source с интеллектом уровня SOTA доступен для тестов
-----
Представлена Kimi K2.5 — новая модель на открытых весах, которая по академическим тестам и качеству instruction following конкурирует с Gemini 1.5 Pro, Claude и ChatGPT (7-е место на LM Arena). 

Модель ориентирована на **Enterprise-сегмент**: обладая 1T параметров, она требует значительных ресурсов (около 600 ГБ VRAM), но предлагает интеллект уровня SOTA. На данный момент Kilo Code открыл бесплатный доступ к модели на одну неделю для публичного тестирования и независимой верификации возможностей.
