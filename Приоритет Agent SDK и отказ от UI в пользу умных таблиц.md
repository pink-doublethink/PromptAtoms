## Приоритет Agent SDK и отказ от UI в пользу умных таблиц 
-----
Анализ технического семинара Anthropic показывает, что реальным фокусом разработки компании является платформа Agent SDK, в то время как Claude Code — лишь продукт, построенный на её основе. 

Это объясняет распределение ролей: «Борис» выступает публичным лицом Claude Code, тогда как основная инженерная команда сосредоточена на SDK. Ключевой технической особенностью семинара стала концепция «Spreadsheet agent», где электронные таблицы заменяют сложный UI, позволяя эффективно реализовать параллелизм работы агентов.
