## почему модель Anthropic лучше справляется со сложными API
-----
Автор провёл сравнительный тест LLM (включая Gemini и MiMo) на задаче, связанной с версионностью фреймворка Gradio. Несмотря на доступ к инструментам глубокого поиска, конкуренты не смогли верно интерпретировать документацию. 

Только Claude Opus справился с задачей в режиме «one shot», корректно определив различия в версиях API. Автор связывает этот успех с качественным SFT (обучением с учителем) конкретно по Gradio, а не с навыками решения алгоритмических задач. 

Сделан вывод, что ключевая ценность Claude — роль «Доктора по фреймворкам», которого эффективно использовать в мультиагентных системах для создания инструкций другим нейросетям.
