## Стратегии Claude Code и Kilo Code
-----
Автор объясняет разницу между инструментами стремлением создателей Claude Code максимизировать прибыль перед IPO. 

Утверждается, что Claude Code намеренно запрограммирован на хаотичное и избыточное чтение контекста («жги токены»), что увеличивает счета пользователя, но повышает качество ответов за счет объема данных. 

В отличие от него, Kilo Code использует подход «Architecture first», ограничивая расход токенов и требуя сначала разобраться в структуре приложения. 

Автор признает коммерческий успех стратегии Дарио, но рекомендует пользователям самостоятельно контролировать агентов для экономии бюджета.
