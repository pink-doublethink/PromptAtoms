## генерация Python-скриптов на лету вместо стандартной навигации по коду
-----
DeepSeek V4 применяет новую парадигму навигации по коду: вместо стандартных инструментов (MCP, графы, grep) модель на лету генерирует короткие Python-скрипты для сканирования контекста. 

Для успешной работы этого алгоритма, которому ИИ был целенаправленно обучен, требуются специальные теги в коде в качестве точек привязки.
