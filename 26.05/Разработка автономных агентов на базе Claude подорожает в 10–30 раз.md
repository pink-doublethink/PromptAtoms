## Anthropic отменяет фикс-тарифы
-----
С 15 июня Anthropic отменяет фиксированные тарифы для программного использования Claude, переводя эти задачи на систему ежемесячных сгораемых кредитов по тарифам API (список затронутых инструментов, таких как Agent SDK и GitHub Actions, подтверждается в файле image_a19918.png). 

Из-за этого нововведения разработка с применением автономных агентов станет в 10–30 раз дороже, что уже вызвало панику в сообществе и, вероятно, спровоцирует массовый отток ИТ-специалистов к альтернативным решениям.
