## Удар по ИИ-агентам:
-----
Ценовая политика Anthropic и регулярные односторонние ухудшения EULA направлены на блокировку автономных и бизнес-агентов (включая Claude Agent SDK) на фиксированных подписках. 

Главная цель компании — радикально сократить неконтролируемое потребление токенов и перевести пользователей на оплату через API. 

Пока эти ограничения не критичны для ИТ-разработки и их можно обойти, но по мере прогнозируемого роста числа событийных ИИ-агентов к концу года проблема серьезно усугубится
