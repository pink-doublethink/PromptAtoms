## Пререлиз GRACE 2.0
-----
Для клиентов консалтинга выпущен пререлиз GRACE 2.0, сфокусированный на улучшенной поддержке китайских ИИ-моделей (DeepSeek V4, GLM-5). 

Главным техническим изменением стал переход на стандарт Doxygen для инлайн-документации Python, который обеспечивает высокую скорость обработки кода и генерирует качественную XML-разметку для агентов. 

Кроме того, для ускорения работы ИИ внедрена система специальных GREP-тегов, позволяющая осуществлять быстрый семантический поиск по множеству модулей, опираясь на естественные рабочие траектории моделей.

https://doxygen.nl/
