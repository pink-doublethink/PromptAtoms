## NVFP4 приходит в ComfyUI на Windows
-----
Нативная поддержка NVFP4 для генерации медиа в ComfyUI теперь доступна на Windows без Docker, обеспечивая ускорение создания видео в 1,9 раза на видеокартах серии RTX 50. 

Для работы требуются обновленные версии Python и CUDA; при этом формат сохраняет высокое качество в простых задачах (Text-to-Video и LLM), но вызывает ожидаемые артефакты в высокоточных процессах (Image-to-Video) из-за фундаментальных ограничений 4-битного представления.
