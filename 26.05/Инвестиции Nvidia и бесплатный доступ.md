## как модель Laguna от экс-CTO GitHub завоевывает разработчиков на Kilo Code
-----
На платформе Kilo Code лидирует DeepSeek V4, однако для работы субагентов и сбора контекста разработчики массово используют быструю и бесплатную модель Laguna. 

Это продукт стартапа Poolside с оценкой в $12 млрд и инвестициями от Nvidia, созданный бывшим CTO GitHub Джейсоном Уорнером.
