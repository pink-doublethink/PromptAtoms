## полнотекстовый поиск (grep) признан лучшей стратегией для кодинг-агентов, обойдя векторные решения
-----
Исследование PwC «Is Grep All You Need?» показало, что полнотекстовый поиск (grep) значительно эффективнее векторного для навигации LLM-агентов по коду, поскольку алгоритмам требуются точные, а не приблизительные совпадения. 

Эти результаты подтверждают правильность архитектурных решений в таких продуктах, как Claude Code от Anthropic и GRACE 2.0, сделавших ставку на классический grep.

  
🔗 https://arxiv.org/html/2605.15184v1
