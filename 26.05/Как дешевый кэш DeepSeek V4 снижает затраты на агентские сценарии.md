## Экономика API
-----
Для многоагентных архитектур DeepSeek V4 остается лидером по стоимости чтения благодаря двадцатикратно более дешевому кэшу. 

При грамотной настройке агенты достигают 90% попадания в кэш (cache hit), что дает существенную экономию, однако во время распродаж ценовой баланс меняется.
