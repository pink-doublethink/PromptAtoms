## Ужесточение доступа к ИИ и проблемы с биллингом
-----
Из-за закрытия Google пробного API и грядущих блокировок западных нейросетей сервисом Open Router для пользователей из РФ, доступ к Gemini значительно усложняется. 

В качестве временного решения предлагается использование Google Vertex AI, что требует привязки надежной иностранной банковской карты для биллинга, включения соответствующих API и создания сервисного аккаунта с генерацией JSON-ключа (предварительно отключив ограничения в политиках организации).
