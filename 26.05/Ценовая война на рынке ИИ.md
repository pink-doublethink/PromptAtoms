## китайские разработчики обрушивают тарифы на API и делают кеш бесплатным
-----
Китайские разработчики инициировали ценовую войну на международном рынке ИИ. 

На фоне радикального снижения стоимости MiMo API — до 99% для кешированных токенов за счет новой оптимизации KV-кеша и на 60–80% для остальных запросов — взимание платы за кеш LLM теряет актуальность. 

Подобный демпинг устанавливает новые стандарты конкуренции, заставляя рынок ориентироваться на экстремально низкие тарифы, сопоставимые с ценами DeepSeek.
