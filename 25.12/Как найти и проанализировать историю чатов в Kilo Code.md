## Как найти и проанализировать историю чатов в Kilo Code
-----
Пользователи Kilo Code могут найти историю переписки с агентом в локальной папке AppData.

В случае некорректной работы расширения рекомендуется загрузить файл api_conversation_history.json в Gemini через AI Studio для проведения глубокого анализа.
