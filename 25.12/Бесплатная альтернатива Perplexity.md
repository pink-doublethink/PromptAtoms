## связка Google AI Studio и Kilo Code для эффективной работы с GitHub
-----
Для бесплатного доступа к Gemini 3 с полноценной интеграцией GitHub автор предлагает использовать связку Google AI Studio и сервиса Kilo Code.

AI Studio ценится за «чистую» сессию без предустановленных системных инструкций, но ограничена в инструментах передачи контекста. 

Эту проблему решает Kilo Code: через модель Grok он агрегирует код и данные из репозитория в единый Markdown-файл. Полученный контекст загружается в AI Studio для решения архитектурных задач, а финальная отладка кода производится обратно в Kilo Code.
