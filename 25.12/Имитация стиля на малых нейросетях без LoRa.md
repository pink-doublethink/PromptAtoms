## метод «словаря фраз» для Qwen 3 14B
-----
Автор экспериментирует с моделью Qwen 3 14B для имитации текстового стиля без использования LoRa. 

Поскольку малые языковые модели (SLM) не способны обобщать литературный стиль через классический few-shot промптинг (для этого требуются модели уровня 200B), предлагается альтернативное решение — использование «словаря фраз».

Метод заключается в обучении модели создавать шаблон ответа с использованием ключей, которые на финальном этапе заменяются на конкретные цитаты с сохранением грамматического согласования. 

Это позволяет SLM, которые «понимают, но не могут сказать», выдавать сложные стилистические результаты.
