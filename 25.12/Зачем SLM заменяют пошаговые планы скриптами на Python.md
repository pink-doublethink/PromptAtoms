## Код как стратегия
-----
Обучение малых языковых моделей (SLM) написанию кода на Python является конкурентной альтернативой стандартному Tool-режиму для выполнения многошаговых задач.

Генерация единого скрипта вместо последовательного плана дает агентам существенные преимущества: возможность проверки синтаксиса, сохранение успешных алгоритмов в RAG для повторного использования и высокую скорость исполнения через интерпретатор. 

На текущем технологическом уровне этот подход позволяет надежно автоматизировать сценарии объемом до 10 шагов.
