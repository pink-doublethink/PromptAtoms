## Почему нейросети игнорируют человеческие ТЗ и учатся программировать друг у друга
-----
Вендоры LLM (на примере Qwen) при обучении моделей программированию полностью отказываются от человеческой документации и ТЗ, считая их несоответствующими реальному коду. 

Вместо этого используется процесс генерации синтетических данных: реальный код из GitHub разбивается на мелкие фрагменты (сниппеты), для которых «большая» LLM создает текстовое описание, а затем заново пишет качественный код.

Таким образом, модели обучаются на материалах, созданных искусственным интеллектом, а не людьми. Для эффективной работы пользователям рекомендуется просить LLM переформулировать задачу своими словами, использовать глубокую декомпозицию и не смешивать человеческий код с машинной генерацией.
