## XML в C# работает лучше классических Summary
-----
Стандарт XML Documentation в C# отлично подходит для семантической разметки ИИ-агентов (таких как GRACE) благодаря поддержке тегов `region`, интеграции с IDE и узнаваемости формата. 

Однако традиционные блоки Summary избыточны, поскольку алгоритмы читают сам код; для ИИ гораздо важнее фиксация целеполагания, ключевые слова и ссылки, дающие максимум семантики при минимальных затратах токенов.
