## Команда xAI поддержала разработку популярного клона Claude Code
-----
Команда xAI предоставила токены Grok разработчику Сигриду Джину для создания Claw Code — успешного клона утекшего Claude Code, который уже собрал более 100 тысяч звезд на GitHub. 

Этот шаг стал ответной реакцией на решение Дарио Амодея отключить Илону Маску доступ к модели Opus.
