## Anthropic ограничивает доступ к Claude Code и монетизирует хаос в разработке
-----
Anthropic отключил доступ к Claude Code для пользователей тарифа Pro, вынуждая клиентов переплачивать. 

Поскольку Claude лучше конкурентов справляется с неструктурированным (Ad Hoc) управлением ИИ-агентами, разработчики, не способные наладить строгую архитектуру кодинга, будут вынуждены платить Anthropic своеобразный «налог на бардак».
