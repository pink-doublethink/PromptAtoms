## Переход Kilo Code на ядро Open Code вернул стабильность 
-----
Новая версия Kilo Code для VS Code демонстрирует высокую стабильность благодаря переходу с устаревшего ядра Cline на Open Code. 

Обновление решает проблему накопленных багов, обеспечивает быструю прямую интеграцию с Gemini API без проксирования, снижает расход токенов примерно вдвое и предлагает гибкое переключение между одиночными и мультиагентными системами.

https://blog.kilo.ai/p/we-completely-rebuilt-the-kilo-vs-code-extension
