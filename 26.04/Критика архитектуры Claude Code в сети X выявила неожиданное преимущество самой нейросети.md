## Критика архитектуры Claude Code в сети X выявила неожиданное преимущество самой нейросети
-----
Пользователи социальной сети X обратили внимание на крайне низкое качество исходного кода инструмента Claude Code. 

Тот факт, что продукт продолжает стабильно работать, опираясь на 500 тысяч строк плохо структурированного кода, рассматривается сообществом как прямое доказательство высокой вычислительной мощности и надежности самой базовой нейросети Claude.
