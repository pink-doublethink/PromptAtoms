## минимум эвристик и ставка на тщательное тестирование
-----
Анализ архитектуры Claude Code показал, что продукт представляет собой простой «толстый клиент» к LLM и MCP без классического сервера. Из 500 тысяч строк кода лишь 1,6% содержат ИИ-эвристики — всю интеллектуальную работу выполняет языковая модель. 

Успех инструмента обусловлен не секретными технологиями или сложными манипуляциями, а тщательным предварительным тестированием, что выгодно отличает подход Anthropic от конкурентов.

https://arxiv.org/abs/2604.14228
