## давление Anthropic на Cursor и ответный альянс Open Code с ChatGPT
-----
Anthropic агрессивно продвигает платформу Claude Code, используя встроенную телеметрию для блокировки сторонних агентов. 

Это спровоцировало конфликт: Open Code призвал разработчиков мигрировать на ChatGPT в рамках нового партнерства, а недавние ограничения Claude в Cursor расцениваются сообществом как давление с целью принудительного перевода пользователей на экосистему Anthropic.
