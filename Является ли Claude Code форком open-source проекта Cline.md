## Является ли Claude Code форком open-source проекта Cline 
-----
Несогласованные заявления сотрудников Anthropic о Claude Code и сохранение кода в закрытом репозитории указывают на отсутствие четкой корпоративной политики. 

Автор текста предполагает, что инструмент изначально создавался не как рыночный продукт, а как внутреннее решение на основе форка open-source проекта Claude Dev (ныне Cline). 

Вероятно, репозиторий остается закрытым, чтобы скрыть факт заимствования, хотя открытый код позволил бы сообществу эффективнее исправлять ошибки.
