## Скорость LLM в разработке
-----
Среди популярных LLM для разработки скорость выше 100 токенов в секунду показывают только Gemini 3 Flash и GLM-5 (на тарифе Max). 

Большинство других моделей, включая Claude и Grok Code Fast, обычно выдают от 35 до 50 токенов в секунду, даже при использовании корпоративных серверов, как в случае с интеграцией через Kilo Code.
