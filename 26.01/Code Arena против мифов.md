## реальная конкуренция LLM во фронтенде и ценовой барьер Claude Opus
-----
Платформа Web Dev Arena обновлена до Code Arena, представив реалистичный тест фронтенда на основе готовых приложений.

Результаты выявили расхождение между популярностью моделей в Telegram и их реальной эффективностью: в бюджетном сегменте GLM и ChatGPT 5.2 High составляют серьезную конкуренцию распространенному Claude Sonnet. 

Техническим лидером остается Claude Opus, однако его высокая цена смещает фокус выбора на более доступные, но неоднозначные альтернативы.
