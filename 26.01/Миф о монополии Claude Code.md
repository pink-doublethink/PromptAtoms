## реальная статистика и фрагментация рынка ИИ-разработки 
-----
Анализ статистики расширений VS Code опровергает маркетинговые заявления о доминировании Claude Code, показывая, что его реальная доля рынка не превышает 10%. Несмотря на формальное лидерство GitHub Copilot (67 млн пользователей), обусловленное предустановками, рынок ИИ-агентов остается крайне фрагментированным.

Профессиональные разработчики активно выбирают альтернативные инструменты, такие как Windsurf и экосистема Cline (около 5 млн пользователей), что указывает на отсутствие явного технологического лидера в отрасли.
