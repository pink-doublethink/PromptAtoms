## Перестановки в Web Dev Arena
-----
Стремительный прогресс новых языковых моделей привел к падению бывших лидеров, таких как Gemini Pro 2.5 и Grok Code Fast, в низ рейтинга Web Dev Arena. 

При этом недорогая модель Claude Haiku демонстрирует эффективность на уровне DeepSeek и Qwen Coder, подтверждая свою недооцененность разработчиками. Хотя mimo-v2-flash и kimi-k2 занимают более высокие позиции, текущий рейтинг не учитывает критические недостатки, например, нестабильность Mimo в длинных сессиях.
