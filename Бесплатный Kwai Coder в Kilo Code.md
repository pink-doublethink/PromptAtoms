## достойный конкурент для нейросетевого программирования
-----
В сервисе Kilo Code открыт бесплатный доступ к Kwai Coder — модели, которая в тестах на программирование и работу агентов показывает результаты на уровне Grok Code Fast. 

Пользователям предлагается оценить новинку и поделиться опытом, так как индустрия нуждается в экономичных моделях среднего уровня для решения архитектурных задач.
