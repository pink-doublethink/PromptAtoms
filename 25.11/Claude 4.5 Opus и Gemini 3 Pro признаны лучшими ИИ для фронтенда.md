На платформе Web Dev Arena модель Claude 4.5 Opus разделила лидерство с Gemini 3 Pro: обе нейросети показали лучшие результаты во фронтенд-разработке, оставив остальных конкурентов далеко позади.

Тесты подтвердили, что для создания интерфейсов стоит выбирать именно эти модели, в то время как для сложных алгоритмов бэкенда эксперты рекомендуют ориентироваться на другие бенчмарки.
