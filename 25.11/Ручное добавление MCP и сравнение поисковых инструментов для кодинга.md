## Прокачка Kilo Code 
-----
В Kilo Code можно вручную подключать любые MCP-серверы, не ограничиваясь встроенным маркетплейсом.

Для задач разработки кода наиболее эффективны поисковые MCP (Tavily, Perplexity, Context 7), позволяющие агентам проверять актуальность API и архитектурные решения. 

Автор рекомендует использовать API Perplexity (включая бюджетные запросы без Deep Search) как наиболее качественный инструмент для анализа источников, отмечая, что Context 7 уступает конкурентам, а Tavily не хватает детальной саммаризации фактов.
