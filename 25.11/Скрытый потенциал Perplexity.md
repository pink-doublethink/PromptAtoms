## бесплатная архитектура и код-ревью через полный доступ к GitHub 
-----
Обнаружено, что GitHub-коннектор в Perplexity обладает полным функционалом, а не только режимом чтения. Это позволяет бесплатно использовать Claude Sonnet для архитектурного проектирования и код-ревью, организуя взаимодействие с кодирующими агентами через Markdown-файлы.

Подход менее удобен для быстрых итераций, но эффективен для фундаментальных задач, где рутину выполняют агенты, а сложные проблемы решаются через запросы к внешним LLM.
