##  успех Gemini, доминирование Opus и «дизайнерский» дефект Codex 5.3
-----
В обновленном рейтинге Code Arena модель Gemini 3.1 Pro улучшила позиции, фактически сравнявшись с GLM-5, в то время как Opus удерживает уверенное лидерство с отрывом в 100 баллов. 

Отсутствие в списке Chat GPT Codex 5.3 автор связывает с вероятными недостатками в эстетике генерируемых интерфейсов, что критически влияет на оценки пользователей в категории Web Dev, в отличие от стандартных алгоритмических бенчмарков.
