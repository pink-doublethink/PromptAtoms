## Сметы против графиков 
-----
Автор описывает проблематику разработки ПО для промышленного строительства на примере интеграции MS Project и сметных данных для крупных заказчиков (вроде «Газпрома»). 

Ключевая сложность заключается в необходимости связать огромные массивы проектно-сметной документации (сотни тысяч операций) с укрупненными календарными графиками, имеющими принципиально иную структуру и логику. 

Традиционные решения этой задачи требуют дорогостоящего ручного сопоставления данных («много-ко-многим») и сложной очистки от ошибок. В связи с высокой стоимостью владения (TCO) текущих подходов, автор анонсирует переработку приложения с использованием искусственного интеллекта.
