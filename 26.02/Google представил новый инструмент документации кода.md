## Google представил новый инструмент документации кода для репозиториев GitHub 
-----
Google запустил систему документации кода, аналогичную CodeWiki, которая уже доступна для многих публичных репозиториев на GitHub. Доступ к функционалу для приватных репозиториев в настоящее время осуществляется через лист ожидания.
