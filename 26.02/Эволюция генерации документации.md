## новый инструмент на GitHub от z ai адаптирует легаси-код для ИИ-агентов
-----
 Команда «Зайцы» представила на GitHub мощный бесплатный генератор документации, который автоматически выстраивает весь пайплайн, включая создание Call Graph . 

 Выпуск этого инструмента делает неактуальной поддержку большинства самописных решений. Генератор особенно критичен для работы со старым (legacy) кодом, где невозможно применить современные практики встроенной документации. 
 
 Сгенерированные материалы отлично подходят для анализа кодовой базы ИИ-агентами. Тем не менее, для обеспечения более эффективного контекста и семантической компрессии, разработчикам рекомендуется самостоятельно дополнять документацию высокоуровневым графом связей сущностей .
