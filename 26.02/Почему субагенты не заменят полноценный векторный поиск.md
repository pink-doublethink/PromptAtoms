## Почему субагенты не заменят полноценный векторный поиск 
-----
Использование субагентов в Claude Code служит вынужденной заменой отсутствующего векторного поиска, позволяя дешевым моделям (типа Haiku) сканировать код.

Автор критикует стратегию Anthropic, называя отсутствие классического векторного RAG серьезным недостатком для работы с крупными кодовыми базами, который лишь частично компенсируется параллельным запуском агентов для исследования структуры проекта.
