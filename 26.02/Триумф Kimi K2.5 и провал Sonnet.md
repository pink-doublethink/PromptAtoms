## как китайские модели меняют расклад сил в Code Arena
-----
Модель Kimi K2.5 возглавила рейтинг Code Arena, став сильнейшим инструментом для кодинга на открытых весах и опередив GLM.

Обновление рейтинга принесло неожиданные результаты: Claude Sonnet опустилась на 14-е место, уступив даже старым версиям DeepSeek, что указывает на её переоцененность. 

Автор считает, что реальную конкуренцию китайским разработкам сейчас может составить только Opus, а грядущий релиз DeepSeek V4 станет серьезным вызовом для Anthropic.
