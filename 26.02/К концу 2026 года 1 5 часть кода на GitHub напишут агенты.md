## К концу 2026 года 1/5 часть кода на GitHub напишут агенты 
-----
Доля коммитов от Claude Code на GitHub уже составляет 4% и при сохранении текущей динамики достигнет 20% к концу 2026 года.

Этот рост знаменует окончание эпохи «вторых пилотов» и переход индустрии к использованию полноценных автономных AI-агентов
