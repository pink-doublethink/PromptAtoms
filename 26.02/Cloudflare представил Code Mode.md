## Песочница для настройки CDN с помощью ИИ-агентов
-----
Cloudflare запустил режим Code Mode, позволяющий ИИ-агентам управлять настройками сети через бесплатную изолированную среду (sandbox). 

Вместо отправки множества API-запросов агенты могут напрямую выполнять JS-код для взаимодействия с 2500 эндпоинтами CDN и самостоятельного создания Worker-скриптов, однако эти возможности еще требуют практического тестирования.

https://blog.cloudflare.com/code-mode-mcp/
