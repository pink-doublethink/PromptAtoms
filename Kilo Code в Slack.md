## бесплатная оркестрация через Claude Opus и фабрика дешевых агентов Grok 
-----
Kilo Code представил интеграцию для Slack, позиционируемую как конкурент Agent SDK. Система использует бесплатный Claude Opus 4.6 в роли оркестратора, который управляет параллельным запуском субагентов на базе Grok 4 Code Fast. Это позволяет эффективно выполнять задачи по анализу репозиториев, тестированию и фиксам с минимальными финансовыми затратами.

https://app.kilo.ai/cloud/sessions
