## Как Python заменяет JSON в стратегиях планирования ИИ-агентов
-----
В сфере долгосрочного планирования ИИ-агентов набирает популярность тренд "Code-as-Plan". 

Суть подхода заключается в том, что вместо традиционных схем (например, JSON) модели генерируют планы своих действий в виде исполняемого кода на Python. 

Это позволяет использовать условные операторы и циклы для гибкого управления рисками и предотвращения зацикливаний. 

Вендоры, включая разработчиков Qwen, активно внедряют эту стратегию, обучая малые языковые модели (SLM) писать код специально для создания сложных, самоуправляемых алгоритмов работы агентов.
