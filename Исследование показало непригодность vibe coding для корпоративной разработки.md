## Исследование показало непригодность vibe coding для корпоративной разработки 
-----
Исследование университетов США показало, что концепция «vibe coding» Андрея Карпатого (автономное написание кода по коротким промптам) эффективна для малых задач, но не масштабируется в enterprise-секторе. 

В реальной корпоративной разработке автономность ИИ оказывается иллюзорной: специалистам приходится жестко контролировать агентов и оперировать большими объемами документации, а не краткими заданиями.


https://arxiv.org/abs/2512.14012
