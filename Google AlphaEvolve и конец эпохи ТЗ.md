## почему для ИИ-агентов код становится важнее документации
-----
Google представил API агента AlphaEvolve, предназначенного не для написания кода с нуля, а для рефакторинга и оптимизации сложных алгоритмов в специфических отраслях (биотех, логистика, финансы). 

Запуск инструмента подтверждает формирование новой парадигмы: ИИ-агенты для отладки опираются на семантику самого кода, а не на внешние технические задания. 

Это смещает фокус разработки в сторону самодокументируемого кода, делая его единственным источником истины и снижая значимость классических спецификаций.
