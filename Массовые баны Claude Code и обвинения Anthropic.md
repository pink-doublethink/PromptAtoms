## массовые баны и обвинения Anthropic в тайной слежке за пользователями
-----
В стремлении повысить прибыльность перед IPO, Anthropic начала массовые блокировки пользователей Claude Code, обвиняя их в нарушении лицензии через шеринг и перепродажу аккаунтов. 

Для выявления нарушений компания, не уведомив клиентов, внедрила скрытую «телеметрию» для сбора данных, на основании которых теперь принимаются односторонние решения о банах.
