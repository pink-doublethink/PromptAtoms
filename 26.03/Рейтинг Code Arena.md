## лидерство Claude и почему системе оценки нужны тесты для бэкенда 
-----
В рейтинге Code Arena уверенно лидирует Claude, за которым с минимальным отрывом следует плотная группа из OpenAI, Google, Z.ai, MiniMax и Moonshot. 

Однако текущая система оценки искажена: она слишком зависит от визуальной привлекательности кода (фронтенда). 

Для повышения объективности платформе необходимо внедрить отдельные тесты для бэкенда, где визуализация будет ограничена, а пользователи смогут оценивать исключительно качество и точность расчетов.
