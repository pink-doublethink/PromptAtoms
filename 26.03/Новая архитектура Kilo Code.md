## отказ от изоляции сред ради параллельной работы роев ИИ-агентов 
-----
Команда Kilo Code переписывает свой продукт с нуля для поддержки параллельной работы роев ИИ-агентов. 

Новая архитектура позволит множеству агентов координированно функционировать в единой тестовой среде без строгой инфраструктурной изоляции (без отдельных виртуальных машин и Docker). 

Это требует специальной адаптации баз данных и микросервисов, но в итоге значительно ускоряет процессы разработки и тестирования.
