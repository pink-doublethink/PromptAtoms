## эталонный шаблон архитектуры репозитория 
-----
Представлена модульная структура файловой системы, оптимизированная для разработки с использованием Claude Code. 

Ключевым элементом является файл `CLAUDE.md` (контекст и инструкции для ИИ), а также специальная директория `.claude` для хранения настроек, хуков и переиспользуемых навыков (skills). 

Архитектура поощряет разделение документации, исходного кода и инструментов автоматизации, что упрощает управление контекстом ИИ и делает рабочие процессы (например, рефакторинг или код-ревью) воспроизводимыми и структурированными.
