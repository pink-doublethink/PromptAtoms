## слабый старт Grok 4.2 и лидерство Claude на фоне равной конкуренции ИТ-гигантов
-----
Обновление платформы Code Arena выявило отставание новой модели Grok 4.2 от xAI, которая оценивается на уровне устаревшего DeepSeek v3.2 и уступает китайским аналогам второго эшелона. 

В задачах на React модели OpenAI, Google, Z.Ai и Moonshot демонстрируют паритет, в то время как Claude сохраняет заметное преимущество, которое отчасти обусловлено субъективным влиянием эстетики дизайна на пользовательское голосование.
