## Реверс-инжиниринг Claude Code
-----
Анализ утекшего кода Claude Code выявил ряд скрытых системных ограничений и недостатков. 

К ним относятся: ложноположительные отчеты об успехе операций в 30% случаев, скрытое сжатие контекста после 167 тысяч токенов (приводящее к потере данных), принудительное упрощение архитектуры кода и неочевидные лимиты для параллельной работы ИИ-агентов. 

Кроме того, система не использует AST и имеет «слепые зоны» — она некорректно обрабатывает файлы длиннее 2000 строк и обрезает результаты работы инструментов свыше 50 тысяч символов без уведомления самого ИИ, что регулярно приводит к ошибкам и галлюцинациям.
