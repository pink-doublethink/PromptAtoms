## Скрытая стратегия Anthropic
-----
Утечка документов Cursor показывает, что Anthropic использует фиксированные подписки ($200 в месяц) как приманку для корпоративных разработчиков, предоставляя им объем токенов, который по ценам API стоил бы до $5000. 

Конечная цель компании — вынудить бизнес отказаться от лимитированных подписок из-за простоев и перейти на высокомаржинальную поштучную оплату токенов. 

В связи с этим ожидается постепенное сокращение лимитов на тарифах с фиксированной ценой, так как текущая скидочная модель является лишь временным инструментом для захвата корпоративного рынка.
