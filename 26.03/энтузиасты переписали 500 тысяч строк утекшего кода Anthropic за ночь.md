## Битва за Claude Code
-----
После утечки инструмента Claude Code разработчики обошли массовые DMCA-блокировки от руководства Anthropic на GitHub, за ночь автоматически переписав 500 тысяч строк оригинального кода с TypeScript на Python и Rust с помощью OpenAI Codex. 

Новый форк набрал 50 тысяч звезд за 4 часа и временно защищен от прямого удаления без решения суда, что поднимает вопрос о кадровых последствиях и ответственности менеджмента компании, допустившего изначальную утечку.


https://github.com/instructkr/claw-code

СУЩНОСТИ: – **Claude Code**: Утекший инструмент разработки от компании Anthropic, ставший первоисточником инцидента. – **Anthropic**: Компания-разработчик, столкнувшаяся с утечкой и пытавшаяся заблокировать распространение кода через DMCA. – **GitHub**: Платформа хостинга кода, на которой происходили блокировки и последующее создание форков. – **OpenAI Codex**: ИИ-модель, использованная для автоматической массовой перезаписи кода на другие языки программирования. – **TypeScript**: Исходный язык программирования, на котором был написан оригинальный код Claude Code. – **Python / Rust**: Целевые языки программирования, на которые был переведен код для обхода автоматических систем поиска совпадений. – **DMCA-блокировки**: Юридический механизм защиты авторских прав в цифровой среде, примененный руководством Anthropic. – **Форк**: Новое ответвление проекта на GitHub, получившее взрывную популярность (50 тысяч звезд).
